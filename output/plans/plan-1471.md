# Plan Fix — Issue 1471
**AuditLog: DocId bị gán sai khi URL có query string; ScreenUrl thiếu điều kiện URL hợp lệ**

## Thông tin
- Branch: `fix/fix-CKTM-1471-tinhlm`
- Base: `release/1.0.0-avntt-rc1`
- Priority: Medium | Type: Issue

## Root Cause

**File:** `src/HQSOFT.Xspire.Application.HttpApi/AuditLogging/ExtendedAuditLogContributor.cs`

**Bug 1 — ScreenUrl:** `urlProp` được check trước khi set ScreenUrl, nhưng lần đầu truy cập (chưa có property) → `urlProp = null` → không set dù `url` có giá trị hợp lệ.

**Bug 2 — DocId:** Dùng `uri.PathAndQuery` (gồm cả `?tab=1&...`) nên `path.Substring(lastIndex + 1)` có thể trả về `"abc123?tab=1"` → gán DocId sai (không phải Guid). DocId phải chỉ được set khi segment cuối URL là Guid hợp lệ.

## Fix

**Bước 1 — ScreenUrl (bỏ check `urlProp != null`):**
```csharp
// SAI:
var urlProp = context.AuditInfo.GetProperty("ScreenUrl");
var url = context.GetHttpContext()?.Request.Headers["screen-url"].ToString();
if (urlProp != null && !string.IsNullOrEmpty(url))
    context.AuditInfo.SetProperty("ScreenUrl", url);

// ĐÚNG:
var url = context.GetHttpContext()?.Request.Headers["screen-url"].ToString();
if (!string.IsNullOrEmpty(url))
    context.AuditInfo.SetProperty("ScreenUrl", url);
```

**Bước 2 — DocId (dùng AbsolutePath, chỉ set khi Guid hợp lệ):**
```csharp
// SAI:
var path = uri.PathAndQuery;
var lastIndex = path.LastIndexOf('/');
var urlDoc = path.Substring(0, lastIndex + 1);
var docId = lastIndex < path.Length - 1 ? path.Substring(lastIndex + 1) : null;
change.SetProperty("ScreenUrl", urlDoc);
change.SetProperty("DocId", docId);  // ← gán kể cả khi là query string

// ĐÚNG:
var path = uri.AbsolutePath;  // ← không chứa query string
var lastIndex = path.LastIndexOf('/');
var urlDoc = path.Substring(0, lastIndex + 1);
var docIdStr = lastIndex < path.Length - 1 ? path.Substring(lastIndex + 1) : null;
change.SetProperty("ScreenUrl", urlDoc);
// Chỉ set DocId khi segment cuối là Guid hợp lệ
if (Guid.TryParse(docIdStr, out var docGuid))
{
    change.SetProperty("DocId", docGuid);
}
```
