## Current Behavior (observed from code)
Khi SAP gửi lệnh hủy đơn hàng DC (WF_DC có CKTM):
1. `ProcessCancelSOAsync` phát hiện `SalesOrderTradeDiscount` records → xóa chúng
2. Kiểm tra `SalesOrderDiscount` (dành cho khuyến mãi KM, không phải CKTM)
3. Nếu `SalesOrderDiscount == null` (đơn DC thuần CKTM không có record này) → **return sớm**
4. `BonusLine.AccruedAmount` KHÔNG bao giờ được giảm → CT CKTM không được hoàn tiền

## Expected Behavior
Khi SAP hủy đơn DC có CKTM:
- `BonusLine.AccruedAmount` phải được giảm bằng tổng `PaymentAmount` của các `SalesOrderTradeDiscount`
- `BonusLine.RemainingAmount` phải được tính lại
- CT CKTM (T_FS20260501) phản ánh đúng số dư còn lại sau khi hủy

## Root Cause
**File:** `modules/hqsoft.sap.dmsintegration/src/HQSOFT.SAP.DMSIntegration.EntityFrameworkCore/CancelSO/EfCoreCancelSORepository.Extended.cs`

Method `ProcessCancelSOAsync` — logic xóa `SalesOrderTradeDiscount` nhưng không hoàn lại `BonusLine`:
```csharp
// BUG: Xóa records nhưng không giảm BonusLine.AccruedAmount
await _salesOrderTradeDiscountRepository.DeleteManyAsync(listSalesOrderTradeDiscount, autoSave: true);

// Sau đó check SalesOrderDiscount — nếu null (DC thuần CKTM), return luôn
if (salesOrderDiscount == null) {
    return response;  // ← BonusLine không được cập nhật!
}
```

## Fix
**Vị trí:** trước `DeleteManyAsync`, thêm loop hoàn `BonusLine` theo từng trade discount (dedup theo `BonusLineId`):

```csharp
// Hoàn AccruedAmount về BonusLine cho từng CKTM trước khi xóa
var dedupedForRefund = listSalesOrderTradeDiscount
    .Where(x => x.BonusLineId.HasValue && x.PaymentAmount > 0)
    .GroupBy(x => x.BonusLineId!.Value)
    .Select(g => g.First())
    .ToList();

foreach (var tdItem in dedupedForRefund)
{
    var bonusLineToRefund = await dbContext.BonusLines.IgnoreQueryFilters()
        .FirstOrDefaultAsync(b => b.Id == tdItem.BonusLineId!.Value);
    if (bonusLineToRefund != null)
    {
        bonusLineToRefund.AccruedAmount -= tdItem.PaymentAmount;
        bonusLineToRefund.RemainingAmount = bonusLineToRefund.TotalAmountIncentive - bonusLineToRefund.AccruedAmount;
        await _bonusLineRepository.UpdateAsync(bonusLineToRefund, autoSave: true);
    }
}
```
