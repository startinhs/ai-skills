# Plan Fix — Issue 1353 / 1354
**ProductGrouping: Hiển thị thông báo thành công không cần thiết khi thêm sản phẩm có trùng**

## Thông tin
- Branch: `fix/fix-SKU-1353-1354-tinhlm`
- Base: `release/1.0.0-avntt-rc1`
- Priority: Low | Type: Issue

## Root Cause

Khi user thêm sản phẩm vào Product Grouping qua popup, code luôn hiển thị cả thông báo **Success** lẫn **Warning** (trùng mã). Điều này gây nhầm lẫn: user thấy "Thêm thành công" dù thực ra toàn bộ sản phẩm bị reject vì trùng.

**File:** `src/.../Pages/MasterData/ProductGrouping/ProductGrouping.razor.cs`

**Trước fix (SAI — hiển thị Success + Warning cùng lúc):**
```csharp
if (duplicateCount > 0)
{
    var duplicateMessage = ...;
    await UiMessageService.Success(L["Đã thêm {0} dòng vào danh sách", addedCount]);  // ← thừa
    await UiMessageService.Warn(duplicateMessage);
}
else if (addedCount > 0)
{
    await UiMessageService.Success(L["Đã thêm {0} dòng vào danh sách", addedCount]);  // ← thừa
}
```

**Sau fix (ĐÚNG — chỉ hiện Warning khi có trùng, bỏ Success messages):**
```csharp
if (duplicateCount > 0)
{
    var duplicateMessage = ...;
    await UiMessageService.Warn(duplicateMessage);  // Chỉ hiện warning
}
// Không hiện success message — grid update là đủ feedback
```

## Fix

**File:** `src/HQSOFT.Xspire.Application.Blazor/Pages/MasterData/ProductGrouping/ProductGrouping.razor.cs`

Trong method xử lý confirm popup thêm sản phẩm:
1. Xóa `await UiMessageService.Success(...)` trong block `if (duplicateCount > 0)`
2. Xóa toàn bộ block `else if (addedCount > 0)` (cũng hiện success)
